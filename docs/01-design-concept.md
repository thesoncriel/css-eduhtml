# CSS.EduHTML.com 디자인 컨셉 문서

## 1. 원본 사이트 분석 (2000~2003년)

### 원본 구조
- **프레임셋 기반**: 좌측 150px 메뉴 프레임 + 우측 본문 프레임
- **IE 5.5 전용**: IE 전용 필터(shadow, glow, alpha) 다수 사용
- **텍스트 그림자**: 동일 텍스트를 약간 offset하여 중첩 배치 (수동 그림자 효과)
- **marquee 태그**: IE 전용 가로 스크롤 애니메이션
- **다채로운 색상**: font 태그로 빨강, 주황, 파랑, 초록 등 다양한 색상 사용
- **이미지 미사용**: CSS의 특성을 보여주기 위해 의도적으로 이미지를 배제

### 원본 색상 팔레트
- 배경: 어두운 계열 (검정/남색)
- 주황(#ff8800): 포인트 색상, 메뉴 아이콘
- 파랑/보라(#8888ff, #aaaaff, #ccccff): 링크, 텍스트
- 빨강(#ff0000, #aa0000): 강조, CSS 키워드
- 초록(#00aaaa, #00cccc): HTML 관련 텍스트

### 원본 페이지 구성
1. **인트로 페이지**: "Cascading Style Sheet" 대형 타이포그래피
2. **메인 페이지**: 환영 인사, NEWS, Free Talk
3. **What's CSS?**: CSS 소개글
4. **Web Studings**: 19개 강좌 목록 (기초/중급/고급)
5. **Download Files**: 특수문자 사전, 그림폰트 사전
6. **Question & Answer**: 제로보드 게시판
7. **Web Site Link**: 추천 사이트 링크

## 2. 모던 디자인 방향

### 디자인 원칙
1. **레트로 감성 보존**: 2000년대 초반의 따뜻한 색감과 운영자의 친근한 어투 유지
2. **모던 레이아웃**: 프레임 → 반응형 사이드바, 테이블 레이아웃 → CSS Grid/Flex
3. **가독성 향상**: 과도한 색상 → 절제된 강조(mark, strong, em)
4. **접근성**: 반응형, 시맨틱 HTML5, 키보드 네비게이션

### 색상 시스템 (TailwindCSS 커스텀)
```
- 배경: slate-900 (#0f172a) ~ slate-950 (#020617)
- 카드/섹션 배경: slate-800 (#1e293b)
- 주 강조색: amber-500 (#f59e0b) - 원본의 주황 계승
- 보조 강조색: indigo-400 (#818cf8) - 원본의 보라/파랑 계승
- 텍스트: slate-100 (#f1f5f9)
- 서브 텍스트: slate-400 (#94a3b8)
- 코드 블록: slate-700 배경, emerald-400 텍스트
- 링크: indigo-400, hover시 indigo-300
- 경고/중요: rose-400 (#fb7185)
```

### 타이포그래피
- 제목: 한글 system-ui, 영문 serif (Times New Roman 감성 계승)
- 본문: system-ui, sans-serif
- 코드: monospace
- 원본의 다양한 폰트 크기 → 체계적인 h1~h6 + body 크기 체계

### 레이아웃
```
┌─────────────────────────────────────────────┐
│  Header (모바일: 햄버거 메뉴)                  │
├──────────┬──────────────────────────────────┤
│          │                                  │
│  Sidebar │        Main Content              │
│  (240px) │                                  │
│          │                                  │
│  - Home  │                                  │
│  - CSS?  │                                  │
│  - Study │                                  │
│  - Down  │                                  │
│  - Q&A   │                                  │
│  - Link  │                                  │
│          │                                  │
├──────────┴──────────────────────────────────┤
│  Footer                                     │
└─────────────────────────────────────────────┘
```

### 반응형 브레이크포인트
- **모바일** (< 768px): 사이드바 숨김, 햄버거 메뉴로 토글
- **태블릿** (768px~1024px): 사이드바 축소 표시
- **데스크톱** (> 1024px): 전체 사이드바 표시

### 주요 변환 규칙
| 원본 기법 | 모던 변환 |
|-----------|-----------|
| 중복 텍스트 그림자 | `text-shadow` (TailwindCSS) |
| `<marquee>` | CSS `@keyframes` 가로 스크롤 |
| `<frameset>` | 사이드바 네비게이션 |
| IE filter:shadow | TailwindCSS `drop-shadow` |
| IE filter:glow | TailwindCSS `shadow` + 색상 |
| `<font color>` 남용 | `<mark>`, `<strong>`, `<em>` + 제한된 색상 |
| 테이블 레이아웃 | CSS Grid / Flexbox |
| `<xmp>` 코드 블록 | `<pre><code>` + 신택스 하이라이팅 |
| `<fieldset>` Free Talk | 모던 카드 컴포넌트 |

### 애니메이션
- 페이지 진입: 부드러운 fade-in
- 사이드바 메뉴: slide-in (모바일)
- Study 헤더: marquee 대체 가로 스크롤 (CSS animation)
- 호버 효과: 부드러운 transition

## 3. 컴포넌트 구조

### 공통 컴포넌트
- `Sidebar`: 사이드 네비게이션 (로고 + 메뉴)
- `MobileNav`: 모바일 햄버거 메뉴 + 오버레이
- `Layout`: 사이드바 + 메인 콘텐츠 래퍼
- `MarqueeText`: marquee 대체 애니메이션 컴포넌트

### 페이지별 특이사항
- **Study 목록**: 기초/중급/고급 섹션 구분 카드
- **Study 상세**: 코드 예제 블록 + 설명 텍스트
- **Q&A**: JSON 데이터 기반 게시판 UI (목록 + 상세)
- **Download**: 다운로드 카드 (설명 + 링크)
